A React + Vite starter template for the Web Development restaurant project.

## Getting Started

1. Copy these files into your GitHub repo folder
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:5173 in Chrome

## What to Customize

Look for `[YOUR RESTAURANT NAME]`, `[TODO]`, and placeholder text throughout the code.
The main files you'll edit:

- `src/components/Navbar.jsx` — Your restaurant name
- `src/components/Footer.jsx` — Your restaurant name and address
- `src/pages/Home.jsx` — Hero section, welcome text, feature cards
- `src/pages/About.jsx` — Your story, team members
- `src/pages/Location.jsx` — Address, hours, map
- `src/pages/Contact.jsx` — Customize text (form already works!)
- `src/App.css` — Change colors to match your plan
- `src/index.css` — Change fonts if desired
- `index.html` — Change page title and Google Fonts link

## Week 3: Firebase

- Edit `src/firebase.js` with your Firebase config keys
- Edit `src/pages/Menu.jsx` — uncomment Firebase code, delete sample data

## Tech Stack

React 18 • Vite • React Router • Firebase Firestore
